You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

function isArmstrong(num: number): boolean {

    const text = num.toString();
    const l = text.length;
    let res = 0;
    for (let digit of text)
        res += parseInt(digit) ** l;
    
    return res == num;
}

Without converting to string create a list of digits by continuously division by 10. 

function isArmstrong(num: number): boolean {

    let temp = num, digits = [];
    while(temp > 0) {
        const digit = temp % 10;
        digits.push(digit);
        temp = Math.floor(temp / 10);
    }
    const l = digits.length;
    
    return digits.reduce((a, b) => a + Math.pow(b, l), 0) == num;
}

Create an array of digits by converting original number to string, splitting it into array of string digits and converting every string digit back into integer. Use reduce to count sum of powered digits. 

function isArmstrong(num: number): boolean {

    const digits = num.toString().split('').map(Number);
    const len = digits.length;
    
    return digits.reduce((sum, digit) => sum + Math.pow(digit, len), 0) === num;
}
