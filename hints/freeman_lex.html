You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def is_armstrong(num: int) -> bool:

    text = str(num)
    l = len(text)
    res = 0
    for digit in text:
        res += int(digit) ** l
        
    return res == num

Without converting to string create a list of digits by continuously division by 10. 

def is_armstrong(num: int) -> bool:

    temp, digits = num, []
    while temp:
        temp, digit = divmod(temp, 10)
        digits.append(digit)
    l = len(digits)

    return sum(d**l for d in digits) == num

Using walrus operator to declare two variables in one line and map function to apply some function to every digit. 

def is_armstrong(num: int) -> bool:

    l = len(text := str(num))
    
    return sum(map(lambda i: int(i)**l, text)) == num
