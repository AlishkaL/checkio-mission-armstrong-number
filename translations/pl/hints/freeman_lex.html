Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

def is_armstrong(num: int) -> bool:

    text = str(num)
    l = len(text)
    res = 0
    for digit in text:
        res += int(digit) ** l
        
    return res == num

Bez konwersji na ciąg znaków proszę utworzyć listę cyfr poprzez ciągłe dzielenie przez 10. 

def is_armstrong(num: int) -> bool:

    temp, digits = num, []
    while temp:
        temp, digit = divmod(temp, 10)
        digits.append(digit)
    l = len(digits)

    return sum(d**l for d in digits) == num

Używając operatora mors do zadeklarowania dwóch zmiennych w jednym wierszu i funkcji map, aby zastosować jakąś funkcję do każdej cyfry. 

def is_armstrong(num: int) -> bool:

    l = len(text:= str(num))
    
    return sum(map(lambda i: int(i)**l, text)) == num
