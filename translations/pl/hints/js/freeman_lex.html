Proszę złapać pomysł 💡! Proszę przekonwertować podaną liczbę na ciąg, sprawdzić jego długość i zadeklarować zmienną dla sumy zasilanych cyfr. Proszę wykonać iterację po ciągu, przekonwertować każdą cyfrę z powrotem na liczbę całkowitą, podnieść ją do odpowiedniej potęgi (długość ciągu) i dodać do wartości zmiennej sumującej. Na koniec proszę zwrócić wynik porównania obliczonej sumy z oryginalną liczbą.

Ok, od czego powinienem zacząć?

Proszę przekonwertować podaną liczbę na ciąg znaków za pomocą metody toString(). 

const text = num.toString();

Proszę sprawdzić długość ciągu znaków za pomocą właściwości length. 

const l = text.length;

Proszę zadeklarować zmienną dla sumy zasilanych cyfr z wartością początkową 0. 

let res = 0;
Co robić dalej?

Iterować po text używając for/of loop. 

for (let digit of text)
    ...;
Nadal potrzebuję podpowiedzi.

Proszę przekonwertować każdą cyfrę z powrotem na liczbę całkowitą za pomocą metody parseInt()... 

parseInt(digit)

... i podnieść ją do potęgi równej długości ciągu l. 

parseInt(digit) ** l
Jak zakończyć wyrażenie?

Wewnątrz pętli proszę zwiększyć wartość res o wynik powyższego wyrażenia. 

res += parseInt(digit) ** l;
Jak dokonać porównania wartości?

Tak, w końcu musi Pan zwrócić logiczny wynik porównania obliczonej sumy zasilanych cyfr i oryginalnej liczby. 

return res == num;
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

function isArmstrong(num: number): boolean {

    const text = num.toString();
    const l = text.length;
    let res = 0;
    for (let digit of text)
        res += parseInt(digit) ** l;
    
    return res == num;
}

Jasne, proszę pobrać!

Bez konwersji na ciąg znaków proszę utworzyć listę cyfr poprzez ciągłe dzielenie przez 10. 

function isArmstrong(num: number): boolean {

    let temp = num, digits = [];
    while(temp> 0) {
        const digit = temp % 10;
        digits.push(digit);
        temp = Math.floor(temp / 10);
    }
    const l = digits.length;
    
    return digits.reduce((a, b) => a + Math.pow(b, l), 0) == num;
}

Proszę utworzyć tablicę cyfr, konwertując oryginalną liczbę na ciąg, dzieląc ją na tablicę cyfr ciągu i konwertując każdą cyfrę ciągu z powrotem na liczbę całkowitą. Proszę użyć reduce, aby policzyć sumę zasilanych cyfr. 

function isArmstrong(num: number): boolean {

    const digits = num.toString().split('').map(Number);
    const len = digits.length;
    
    return digits.reduce((sum, digit) => sum + Math.pow(digit, len), 0) === num;
}
