In der Zahlentheorie ist eine Armstrong Zahl (nach Michael F. Armstrong) oder eine narzisstische Zahl in einer bestimmten Zahlenbasis (10 für diese Mission) eine Zahl, die die Summe ihrer eigenen Ziffern ist, die jeweils zur Potenz der Anzahl der Ziffern erhöht werden. Zum Beispiel ist 153 eine Armstrong Zahl, weil 13 + 53 + 33 == 153.
