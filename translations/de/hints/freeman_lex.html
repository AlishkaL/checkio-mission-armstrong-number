Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur alle vorherigen Hinweise zusammenführen und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def is_armstrong(num: int) -> bool:

    text = str(num)
    l = len(text)
    res = 0
    for digit in text:
        res += int(digit) ** l
        
    return res == num

Erstellen Sie ohne Konvertierung in einen String eine Liste von Ziffern durch kontinuierliche Division durch 10. 

def is_armstrong(num: int) -> bool:

    temp, digits = num, []
    while temp:
        temp, digit = divmod(temp, 10)
        digits.append(digit)
    l = len(digits)

    return sum(d**l for d in digits) == num

Verwenden Sie den Walross-Operator, um zwei Variablen in einer Zeile zu deklarieren und die Map-Funktion, um eine Funktion auf jede Ziffer anzuwenden. 

def is_armstrong(num: int) -> bool:

    l = len(text:= str(num))
    
    return sum(map(lambda i: int(i)**l, text)) == num
