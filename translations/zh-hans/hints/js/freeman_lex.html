以下是整个解决方案：

function isArmstrong(num: number): boolean {

    const text = num.toString();
    const l = text.length;
    let res = 0;
    for (let digit of text)
        res += parseInt(digit) ** l;
    
    return res == num;
}

在不转换为字符串的情况下，通过连续除以 10 创建一个数字列表。 

function isArmstrong(num: number): boolean {

    let temp = num, digits = [];
    while(temp> 0) {
        const digit = temp % 10;
        digits.push(digit);
        temp = Math.floor(temp / 10);
    }
    const l = digits.length;
    
    return digits.reduce((a, b) => a + Math.pow(b, l), 0) == num;
}

通过将原始数字转换为字符串，将其拆分为字符串数字数组，再将每个字符串数字转换回整数，从而创建数字数组。使用reduce 计算供电数字的总和。 

function isArmstrong(num: number): boolean {

    const digits = num.toString().split('').map(Number);
    const len = digits.length;
    
    return digits.reduce((sum, digit) => sum + Math.pow(digit, len), 0) === num;
}
