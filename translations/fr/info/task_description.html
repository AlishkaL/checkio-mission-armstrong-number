En théorie des nombres, un nombreArmstrong (après Michael F. Armstrong) ou nombre narcissique dans une base de nombres donnée (10 pour cette mission) est un nombre qui est la somme de ses propres chiffres élevés à la puissance du nombre de chiffres. Par exemple, 153 est un nombre Armstrong car 13 + 53 + 33 == 153.
